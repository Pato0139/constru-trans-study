# Constru-Trans Study Lab

Plantilla de estudio interactiva para aprender el proyecto Constru-Trans.

## 📂 Estructura del Proyecto
```
.
├── index.html      # Estructura HTML
├── styles.css      # Estilos CSS
└── script.js       # Lógica JavaScript
```

## 🚀 Cómo usar
1. Abre `index.html` en tu navegador
2. Sigue las lecciones en orden
3. Completa los ejercicios
4. Marca las lecciones como completadas

## 🎨 Temática
- Colores de zorro: negro y naranja
- Diseño moderno y limpio
- Progreso guardado en localStorage

- Inicio y bienvenida
